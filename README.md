🚀 Tesla Stock Price Predictor

This project is a Machine Learning-based web application that predicts Tesla stock prices using historical data.

---

📌 Project Overview

The goal of this project is to analyze past stock price data and build a predictive model that estimates future stock prices based on previous trends.

---

🧠 Features

- Predict stock price using previous close value
- Simple and user-friendly interface
- Built using Machine Learning
- Real-time prediction through web app

---

🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

📊 Dataset

- Tesla historical stock price dataset
- Data includes Date, Open, High, Low, Close, Volume

---

⚙️ How It Works

1. Load historical stock data
2. Preprocess and clean data
3. Create features (Previous Close Price)
4. Train Machine Learning model
5. Predict future stock prices

---

▶️ How to Run

pip install -r requirements.txt
streamlit run app.py
